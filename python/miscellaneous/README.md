# Introduction
This directory contains some `.ipynb` files that helps to covert model/Classifier(generated using sklearn) into `.java` file.
### How to run
* Run this `.ipynb` file using Google colab to genarate `.java` file from RandomForestClassifier or SVM Classifier.
* For generating `.java` file from Classifier `sklearn-porter` module have been used.

### Reference
* Colab - https://colab.research.google.com/notebooks/intro.ipynb
* sklearn-porter github repo - https://github.com/nok/sklearn-porter
* A sample/demo project using sklearn-porter - https://github.com/tamojit2000/Android-and-Sklearn-porter
